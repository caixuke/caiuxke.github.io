微信授权登录接口

Oct 16 Comments

微信授权登录接口

注册

获取验证码

目前有4种情况，对应的流程图如下

接口参数

  • Method: POST
  • URL: /token/login
  • Params：
    参数名 数据类型 是否必填 说明
    client_id string Y 客户端名称
    grant_type string Y 授权类型，分implicit和password两种
    code string N 获取微信openId所需的code
    open_id string N 微信openId，code和open_id两者必传一个
    username string N 当授权类型grant_type为password时必传
    password string N 当授权类型grant_type为password时必传

Response

  • Body
  1. 当需要使用用户名密码登录时，前一步获得的openId返回在body的redirectUrl的字段里
  2. code对应关系如表，code为1000时表示成功
    code message
    1000 用户通过用户名密码授权绑定账号成功
    1000 用户已授权绑定账号
    1000 用户通过微信认证授权绑定账号成功
    1001 openId获取失败
    1002 用户未授权登录，请先通过用户名密码登录
